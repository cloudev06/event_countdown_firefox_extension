# Firefox Event Countdown Extension

A simple Firefox extension to track countdowns to your favorite events.

## Installation Instructions

1.  **Open Firefox** and navigate to `about:debugging`.
2.  Click on **"This Firefox"** in the left sidebar.
3.  Click the **"Load Temporary Add-on..."** button.
4.  Navigate to this folder:
    `C:\Users\jadee\.gemini\antigravity\scratch\firefox-event-countdown`
5.  Select the `manifest.json` file.

## Usage

1.  Click the extension icon in your toolbar (it looks like a purple hourglass).
2.  Enter an **Event Name** (e.g., "Holiday").
3.  Pick an **Event Date**.
4.  Click **"Add Event"**.
5.  The event will be saved and show the number of days remaining!

## Files

-   `manifest.json`: Configuration file for the extension.
-   `popup.html`: The layout of the popup window.
-   `popup.css`: Styles for the popup.
-   `popup.js`: Logic for handling events and storage.
-   `icons/`: Contains the extension icon.
